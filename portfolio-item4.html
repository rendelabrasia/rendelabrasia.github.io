State Machine

A state machine can change from one state to another based on the the input response. In this case, the list of states are based off my student number. This project has been created in a Linux environment called VNC and C language is used to make the code.

Input 1 shows the starting point being moved to another state while input 0 stays at the same state. Command "p" prints out the list of states.
